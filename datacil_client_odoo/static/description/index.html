Datacil Client

Valida cedulas y RUC, autocompleta datos del cliente con un solo clic

Automatiza la validacion de tus clientes

Integra los servicios de Datacil directamente en tu formulario de contacto. Ingresa una cedula o RUC y el sistema valida la identificacion y completa automaticamente los datos del cliente: nombre, direccion, ciudad, correo y telefono.

Caracteristicas Principales

🔍

Validacion de Identificacion

Valida cedulas (10 digitos) y RUC (13 digitos) de Ecuador en tiempo real usando la API de Datacil.

⚡

Autocompletado de Datos

Completa automaticamente nombre, direccion, ciudad, correo electronico y telefono del cliente.

📊

Dashboard de Monitoreo

Visualiza creditos disponibles, historial de transacciones y costos de cada servicio en tiempo real.

Como Funciona

1 Configura tu API Key
Ve a Ajustes → Datacil e ingresa tu clave de API proporcionada por Datacil.
2 Ingresa la Identificacion
En el formulario de contacto, escribe la cedula o RUC del cliente en el campo de identificacion.
3 Valida con un Clic
Presiona el boton de validacion y los datos del cliente se completaran automaticamente.

Mas Funcionalidades

Deteccion de Duplicados

El sistema detecta si un cliente ya existe en tu base de datos para evitar registros duplicados.

Configuracion por Empresa

Cada empresa puede tener su propia clave de API y configuracion independiente.

Control de Acceso

Grupo de seguridad dedicado para controlar quien puede acceder al dashboard de Datacil.

Manejo de Errores

Mensajes claros cuando la identificacion es invalida, no se encuentra o hay problemas con el servicio.

Configuracion

Accede a la configuracion desde Ajustes → Datacil donde podras establecer:

  • URL de la API: Endpoint del servicio de Datacil
  • Clave de API: Token de autenticacion Bearer
  • Version de API: Version del servicio a utilizar
  • Pais: Pais de validacion (Ecuador)
  • Tiempo de espera: Timeout para las solicitudes a la API
  • Cargar contactos existentes: Permite cargar datos de clientes ya registrados

Requisitos

Dependencias: Este modulo requiere los modulos contacts y l10n_latam_base instalados. Ademas, necesitas una cuenta activa en datacil.com con creditos disponibles para usar el servicio de validacion.
